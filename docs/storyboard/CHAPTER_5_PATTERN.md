# CHAPTER 5 — The Pattern Wakes (W029–W038)

**Arc:** The Pattern (the universe becoming self-aware) bleeds into Cal's visor. Mid-game scope expansion:
bigger worlds, the Signal's consequences visible everywhere. RILL warns the Pattern is spreading toward
*memory*. Ends near the first Warden Standoff (W037). **RILL state:** Unsealing (full vocabulary). **Signal:**
Threshold 3 → climbing to MAX. **Reveal:** "it's waking up — and so are you." Inherit `STORY_BIBLE.md`.

> Visual motif: from here, a faint **Pattern overlay** can bleed into the HUD in any world (rises with Signal).
